# 文字冒险游戏 Text Adventure Game

一个基于 Python 的轻量级文字冒险游戏，支持房间移动、物品收集、背包系统等功能。

## 系统架构图
┌─────────────┐         ┌─────────────┐         ┌─────────────────┐
│  Room 房间类 │◄───────►│ Player 玩家类 │◄───────►│  主游戏循环 main │
└─────────────┘         └─────────────┘         └─────────────────┘

- Room：管理房间描述、物品、出口
- Player：管理当前位置、背包物品
- main：命令解析、游戏交互

## 本地开发环境搭建
1. 安装 Python 3.8+
2. 克隆项目
   git clone https://github.com/sevencp11/team-agile-project.git
3. 运行游戏
   python main.py
4. 运行单元测试
   python test_game.py

## 核心业务模块职责
### Room 类
表示游戏中的房间，包含：
- 房间名称
- 房间描述
- 房间内物品
- 出口方向

### Player 类
表示玩家，包含：
- 当前所在房间
- 背包物品列表

### 游戏命令
- look：查看当前房间
- move [方向]：移动
- take [物品]：拾取物品
- inventory：查看背包
- exit：退出游戏
